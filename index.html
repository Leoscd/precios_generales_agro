📖 Cómo usar esta app en GitHub Pages

  1. Creá un repositorio en GitHub y subí estos archivos: index.html, Repositorio_Precios.xlsx, data/historial.json
  2. Activá GitHub Pages en Settings → Pages → Branch: main
  3. Copiá la URL raw del Excel (botón "Raw" en GitHub) y pegala abajo en "URL del Excel en el repo"
  4. Cuando actualices precios: editá el Excel, subilo al repo. La app detecta los cambios automáticamente
  5. El historial se guarda en tu navegador. Exportalo a JSON y subilo al repo para no perderlo
